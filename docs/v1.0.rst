Introduction
------------

This project allows to create an overloaded function around a core choosing function.

Installation
------------

To install overloadable, you can use pip. Open your terminal and run:

.. code-block:: shell

    pip install overloadable

Features
--------

``overloadable.overloadable``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

``overloadable.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its test function.

.. code-block:: python

    import overloadable
    overloadable.test()
